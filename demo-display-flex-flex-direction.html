CSS Flexible Box Module

flex-direction

Demonstration of a "flex-direction". This also compares with non-flex containers and effects on various flex-items such as block and inline items in each.

The flex direction default is 'row' ("flex-direction: row;"). Use the navigation below to toggle between row and column directions to see how:

  • the main axis flex-direction justification (kept at default: flex-start) aligns items to the start line,
  • the difference between 'flex-direction: row' and 'flex-direction: column' layout.
  • when the dir is rtl, the justification and/or flex-start line changes
