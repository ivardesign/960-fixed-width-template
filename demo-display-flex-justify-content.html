CSS Flexible Box Module

justify-content

There are 10 possible values for justify-content (support varies).

  • flex-start (default)
  • flex-end
  • start
  • end
  • left
  • right
  • center
  • space-between
  • space-around
  • space-evenly

These can be paired with one of two additional keywords:

  • safe
  • unsafe
