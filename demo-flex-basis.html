CSS Flexible Box Module

flex-basis

The flex-basis item property defines a default size for an element before the remaining space is distributed. The size can be a keyword (e.g.: auto, content, fit-content, etc.) or a length (such as 50%, 40em, 100px). Keyword support can be poor.

If flex-basis is set to zero, extra space around content isn't factored in. If set to auto, extra space is distributed based on its flex-grow value.
