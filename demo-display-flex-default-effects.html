Fexible Box Module

"display: flex;"

Demonstration of a flex container with default settings; no flexbox declarations other than 'display: flex'. This also compares with non-flex containers and effects on various flex-items such as block and inline items in each.

In these English (left-to-right, top to bottom flow) demonstrations the default behavior shows how:

  • the main axis flex-direction (default: row) justification (default: flex-start) aligns items to the left edge (the start line),
  • the flex-items can shrink in width compared with non-flex containers, where a DIV will be 100% width,
  • flexible box module honors whitespace when between alpha-numeric characters,
  • inherited margins from the P elements are kept,
  • flex-items can break out of their container when the container is too narrow.
